# MasterCheff - AI Chef Battle

A multiplayer mobile cooking game where players compete by writing dish descriptions that are judged by AI.

## Features
- 4 Player PvP matches
- 10 Rounds per session (~12 minutes)
- AI-powered judging with 3 unique personas
- AI-generated dish images for winning dishes
- Power-ups system

## Tech Stack
- Unity 2022 LTS
- Photon PUN2 for multiplayer
- OpenAI GPT-4o for judging
- DALL-E 3 for image generation
