# Decorative & VFX Elements

Place effect sprites here (various sizes PNG with transparency):

| File Name | Size | Description |
|-----------|------|-------------|
| confetti_particle.png | 16x16 | Small confetti piece for celebrations |
| sparkle.png | 32x32 | Sparkle effect for legendary ingredients |
| trophy.png | 128x128 | Match winner trophy |

| ribbon_silver.png | Variable | 2nd place ribbon |
| ribbon_bronze.png | Variable | 3rd place ribbon |

## Usage

- **confetti_particle.png:** Used in Unity particle system for winner celebrations
- **sparkle.png:** Animated overlay on legendary/rare ingredients
- **trophy.png:** Displayed for match winner
- **ribbons:** Displayed next to player rankings

## Particle System Tips

For confetti, consider creating a sprite sheet with multiple confetti shapes/colors,
or provide several individual confetti pieces (confetti_1.png, confetti_2.png, etc.)
