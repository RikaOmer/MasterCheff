## Orientation

The game supports both orientations. Consider:
- **Portrait (1080x1920):** Mobile-first design
- **Landscape (1920x1080):** Tablet/desktop view

You can provide both versions with suffixes:
- bg_menu_portrait.png
- bg_menu_landscape.png

## Design Guidelines

- Use warm kitchen colors (reds, oranges, browns)
- Keep busy areas at edges, leave center clear for UI
- Consider adding subtle blur/gradient overlays for text readability
- Match lighting across all backgrounds for consistency
