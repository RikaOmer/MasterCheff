# Power-Up Icons

Place 3 power-up icons here (128x128 PNG with transparency):

| File Name | Power-Up | Description |
|-----------|----------|-------------|
| reroll_pantry.png | Reroll Pantry | Dice or refresh icon - allows player to reroll ingredients |
| time_extension.png | Time Extension | Clock/hourglass with plus sign - adds extra cooking time |
| secret_ingredient.png | Secret Ingredient | Mystery box or star icon - add an extra ingredient to dish |

## Design Guidelines

- Use bold, recognizable icons
- Consider adding a glow or special effect for visibility
- Icons should be easily distinguishable at small sizes
- Match the overall game art style
