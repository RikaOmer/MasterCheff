# Rarity Indicators

Place rarity badge sprites here (32x32 PNG with transparency):

| File Name | Rarity | Color | Description |
|-----------|--------|-------|-------------|
| rarity_common.png | Common | Gray/White | Standard ingredients |
| rarity_uncommon.png | Uncommon | Green | Slightly special ingredients |
| rarity_rare.png | Rare | Blue | Hard to find ingredients |
| rarity_legendary.png | Legendary | Gold/Purple | Premium, ultra-rare ingredients |

## Design Guidelines

- Small corner badges that overlay ingredient icons
- Consider adding a glow effect for legendary
- Keep text/icons simple and readable at small sizes
- Can use stars, gems, or simple color badges

## Example Designs

- Simple colored circles with rarity level number (1-4)
- Star badges (1 star, 2 stars, 3 stars, crown/star)
- Gem icons in corresponding colors
- Border glow effects
