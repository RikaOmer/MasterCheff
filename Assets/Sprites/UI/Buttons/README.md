# UI Buttons

Place button sprites here (9-slice compatible PNG):

| File Name | Size | Description |
|-----------|------|-------------|
| button_primary.png | 9-slice | Main action button (Submit, Ready, Start) |
| button_secondary.png | 9-slice | Secondary button (Cancel, Back) |
| button_icon.png | 64x64 | Small icon button (Copy, Settings) |

## 9-Slice Setup

For stretchable buttons, create sprites with:
- Fixed corner areas that don't stretch
- Middle areas that tile/stretch

In Unity, set up 9-slice borders in the Sprite Editor after import.

## Color Suggestions

- **Primary Button:** Warm orange/red - action color
- **Secondary Button:** Neutral gray/brown - less emphasis
- **Icon Button:** Match panel color with subtle border
