# Style Tag Icons

Place 5 style tag icons here (64x64 or 128x128 PNG with transparency):

| File Name | Style | Description |
|-----------|-------|-------------|
| homey_comfort.png | Homey Comfort | Home/heart icon - cozy comfort food |
| gourmet.png | Gourmet Fine Dining | Chef hat/fork icon - upscale cuisine |
| dessert.png | Decadent Dessert | Cupcake/cake icon - sweet treats |
| healthy.png | Healthy Fresh | Leaf/salad icon - light and fresh |
| fusion.png | Crazy Fusion | Mix/explosion icon - wild combinations |

## Design Guidelines

- Each icon should clearly represent its food style
- Use consistent line weights and colors
- Icons appear on buttons players select to tag their dish style
- Make them visually distinct from each other
